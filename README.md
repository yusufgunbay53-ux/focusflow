# FocusFlow

AI Destekli Görev ve Odaklanma Asistanı

Modern, karanlık tema, Kanban Board, Pomodoro Timer ve AI Koç ile odaklanmayı zirveye taşıyan web uygulaması.

## Özellikler
- Drag & Drop Kanban Board
- Gelişmiş Pomodoro Sayacı
- AI Performans Koçu
- Lo-Fi Müzik Player
- localStorage persistence

## Kurulum

```bash
git clone https://github.com/yusufgunbay53-ux/focusflow.git
cd focusflow
npm install
npm run dev
```
